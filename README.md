# BMI-Calculator

A simple BMI Calculator written in Java.

![BMI calculator](https://github.com/user-attachments/assets/a85edcca-5355-4b11-b453-9026b96ec020)

## Requirements

- Java 8 or higher

## How to use

1. Build the project with your preferred build tool or IDE. If a packaged JAR is available in the `target/` folder, you can run it directly.

2. From the command line (replace `<jar-file>.jar` with the actual filename found in `target/`):

```bash
cd target
java -jar <jar-file>.jar
```

3. Or open the project in an IDE (IntelliJ IDEA, Eclipse, etc.) and run the main class.

## Notes

- Make sure Java is installed and available on your PATH.
- If you need help building or running the project, open an issue.
